# Mémoire collective

Mémoire Collective est une plateforme citoyenne indépendante, dédiée à la transparence et à la responsabilisation des politiques. L'objectif est simple : briser le cycle de l’impunité en documentant, archivant et rendant accessibles les manquements des figures publiques, afin que chaque citoyen puisse juger en connaissance de cause.
